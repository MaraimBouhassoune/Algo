PARTIE 3 : CONCLUSIONS ET RECOMMANDATIONS
---------------------------------------------

RECOMMANDATIONS POUR LES TRIS :
• Petits datasets (< 100) : Tri insertion (simple et efficace)
• Datasets moyens/grands   : Tri fusion (O(n log n) garanti, stable)
• Performance pure         : Tri rapide (en moyenne le plus rapide)
• Stabilité requise        : Tri fusion (préserve l'ordre des égaux)

RECOMMANDATIONS POUR LES RECHERCHES :
• Recherche unique sur données triées : Recherche binaire
• Recherche multiple/filtrée          : Recherche linéaire
• Statistiques (min/max)               : Un seul parcours optimisé
• Application réelle                   : Index de base de données
